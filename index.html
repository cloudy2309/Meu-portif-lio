Minha Biblioteca: Uma Webpage Personalizada

Este projeto é uma página web que apresenta uma lista de livros focado no ensinamento de programação e seus conceitos.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro é destacado em seu próprio contêiner (.livro), mostrando uma imagem da capa, detalhes do livro e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração de livros.

Minha Biblioteca: Uma Webpage Personalizada

Este projeto consiste em um site com foco na assinatura da plataforma Alura, voltado para apresentar seus cursos, benefícios e planos de acesso. O objetivo é criar uma página atraente e funcional que convença o usuário a realizar a assinatura, destacando os diferenciais da Alura, como a variedade de cursos (programação, design, data science, etc.), trilhas de aprendizagem, certificações e suporte da comunidade.

É um projeto ideal para treinar a criação de landing pages (páginas de conversão) com foco em marketing digital, design responsivo e experiência do usuário.

HTML: Para estruturar o conteúdo da página — textos, botões, seções, imagens e links. CSS: Para personalizar o estilo visual, cores da marca, espaçamento, tipografia e responsividade. JavaScript: Para interatividade — como animações, sliders de depoimentos, validação de formulários e interações com o botão de assinatura. Formulários e integração com serviços externos (opcional): Para integrar o botão de assinatura com links reais ou plataformas de pagamento (como Stripe, PayPal ou gateways próprios da Alura, se for uma simulação). Git e GitHub (opcional): Para controle de versão, backup e publicação do site (como por GitHub Pages). Noções de UX/UI Design: Importantes para otimizar a navegação e guiar o usuário até a ação principal: assinar a plataforma. Bibliotecas e frameworks (opcional): Bootstrap: para criar um layout limpo e responsivo rapidamente. AOS (Animate on Scroll): para animações suaves ao rolar a página.
